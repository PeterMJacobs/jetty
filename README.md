# jetty
Jets with pythia &amp; fastjet &amp; root sandbox

# requires

 - Installed root and Pythia8 - `pythia8-config` and `root-config` should be in the $PATH.
 - BOOST

# installation


```
$ source <dir_where_downloaded>/scripts/setenv.sh
$ build_jetty.sh
```

# example

$  jettyExamplesExe --pythia
